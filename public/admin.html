Autodesk Build Integration

Configure credentials below to enable push to Autodesk Build.

How to Push to Autodesk Build

  1. Create an app at aps.autodesk.com and enable the Data Management API.
  2. Copy the Client ID and Client Secret into the fields above.
  3. Find your Project ID in Autodesk Build > Project Settings.
  4. Find your target folder URN using the APS Explorer or Data Management API.
  5. After uploading, open Autodesk Build > Schedule > Work Plans and import the uploaded file.
